# Personal Portfolio Website

## Project Overview

This project is a simple Persoafterbnal Portfolio Website developed using HTML5. The website introduces the user, displays skills, and provides a contact form for communication.

### Objectives

* Learn HTML5 structure
* Use semantic HTML tags
* Create navigation between sections
* Add images and links
* Build a contact form

---

## Technologies Used

* HTML5
* Visual Studio Code
* Web Browser
